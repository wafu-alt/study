# NodeJS VS Javascript 차이

## Javascript는.
스크립트 언어라서 특정 프로그램 위에서만 동작을 할 수 있다.   
그것이 웹 브라우저이고, 웹 브라우저 없으면 동작을 못하므로 nodeJS라는 프로그램을 만든 것이다.

## NodeJS는.
터미널 프로그램 언어이고, 자바스크립트로 만들어졌다.

## 자바스크립트가 실행되는 브라우저, 터미널를 런타임이라고 한다.
