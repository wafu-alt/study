문자열.charCodeAt(위치)

```jsx
let strName = "a";
console.log(strName.charCodeAt(0)); // 97
strName = "Aa";
console.log(strName.charCodeAt(0)); // 65
strName = "Aa";
console.log(strName.charCodeAt(1)); // 97

```
